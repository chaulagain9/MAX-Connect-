Code is written in java and have three files which needed to be compiled and run later with input. 

To run this program first of all compile  as follows

javac maxConnect.java AiPlayer.java GameBoard.java

This compilation will create the class files and to run these files enter the following commands. 

Interactive mode 

 1)When  HUMAN plays first
java maxConnect interactive input1.txt human-next 10

2)When COMPUTER plays next
java maxConnect interactive input1.txt computer-next 10

One-move mode

java maxConnect one-move input1.txt output1.txt 10
